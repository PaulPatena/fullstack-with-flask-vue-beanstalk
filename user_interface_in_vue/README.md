## fullstack-with-flask-vue-beanstalk

### Background
This subproject contains the VueJS + Vuetify code that makes up our demo UI. End-to-end testing using Cypress.

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```
